# 余下

一个轻量的个人资金项目记账 PWA，数据只保存在当前设备浏览器中。

支持多个独立资金项目、项目归档与恢复、消费分类统计、每日趋势，以及按项目统计币种统一折算的多币种消费记录。

项目统一以人民币统计，消费支持人民币、美元和印尼盾。当前固定参考汇率为 `1 USD = 6.81 CNY`、`100,000 IDR = 37.60 CNY`，记账时仍可手动调整折合金额。

每个项目可以设置默认支付币种，首页提供快速货币换算入口。

## 本地运行

```bash
python3 -m http.server 4173
```

然后打开 `http://localhost:4173`。

## iPhone 安装

用 Safari 打开正式 HTTPS 地址，点击“分享”并选择“添加到主屏幕”。

清除 Safari 网站数据会删除本地账目，重要数据请先在“设置”中导出备份。
